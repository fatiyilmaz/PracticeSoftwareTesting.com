#  PracticeSoftwareTesting.com - API Automation Test Suite

##  Technologies & Tools

- Java
- Maven
- RestAssured (for API testing)
- Cucumber (BDD-style scenarios)
- JUnit (assertions)
- Jackson (JSON binding)
- Git & GitHub

###  Prerequisites

- Java JDK installed (Java 8+)
- Maven installed
- Internet access to connect to the API
